# ArchiveCore

# Installer les dépendances

```bash

# npm 
npm install

# yarn
yarn install

#pnpm
pnpm install

#bun
bun install
```

# Serveur de développement

```bash

# npm
npm run dev

# yarn
yarn dev

# pnpm
pnpm dev

# bun
bun dev
```
